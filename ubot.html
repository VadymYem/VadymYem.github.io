AuthorBot

✍️ AuthorBot — це сучасний юзербот для Telegram, який працює від імені звичайного користувача, надаючи розширені можливості, недоступні для класичних ботів. Наш юзербот підтримує встановлення на різні платформи: Android через Termux, Windows через WSL, та Linux VDS/VPS сервери.


⁉️ Юзербот — це бот, який працює від імені звичайного користувача Telegram, надаючи розширені можливості, недоступні для звичайних користувачів. Наприклад, юзербот може завантажити відео з тіктоку, бути ботом-модератором в чаті, знайти улюблену пісню, шукати інфо в гугл та виконувати інші дії в улюбленому месенджері без необхідності виходити або запускати якихось інших ботів чи програми. По ідеї працює як і звичайний бот, але від імені користувача.

Встановлення



📱 Termux (Android):

Встановіть Termux за посиланням вище та виконайте наступну команду:

termux-wake-lock && export AIOHTTP_NO_EXTENSIONS=1 && pkg upgr -y && pkg i wget ncurses-utils python openssl git -y && clear && . <(wget -qO- https://raw.githubusercontent.com/VadymYem/AuthorBot/refs/heads/master/termux.sh)

⚠️ Якщо побачите попередження де треба підтвердження, то виконайте: Y . Це треба для успішного встановлення всих необхідних пакетів.
Якщо при виконанні скрипту пише Aborted, то з команди прибираємо `termux-wake-lock &&` і встановлюємо бота

🪟 Windows (через WSL):

Для встановлення на Windows потрібно виконати наступні кроки:

  1. Встановіть WSL:

    Для цього відкрий PowerShell с правами адміністратора і напиши:

    wsl --install -d Ubuntu-22.04

    ⚠️ Потрібна Windows 10 build 2004 або новіше, з підтримкою віртуалізації. Для старіших версій дивіться інструкцію

  2. Перезавантажте ПК та запустіть Ubuntu 22.04
  3. Встановіть pip:
    curl -Ss https://bootstrap.pypa.io/get-pip.py | python3

    ⚠️ Якщо побачите жовті попередження, виконайте: export PATH="/home/username/.local/bin:$PATH" (замініть шлях на вказаний у попередженні)

  4. Встановіть AuthorBot:
    clear && git clone https://github.com/VadymYem/AuthorBot && cd AuthorBot && pip install -r requirements.txt && python3 -m hikka

-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_-_


🌐 VDS/VPS (Linux):

Для встановлення на VDS/VPS сервер виконайте команду:

sudo apt update && sudo apt upgrade -y && sudo apt install git python3 -y && git clone https://github.com/VadymYem/AuthorBot && cd AuthorBot && pip install -r requirements.txt && python3 -m hikka --no-web
